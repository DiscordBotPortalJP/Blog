# ソースコード

`message.channel` が `discord.TextChannel` のインスタンスかどうかで分岐するようにします。

```python
@client.event
async def on_message(message):
    if not isinstance(message.channel, discord.TextChannel):
        return
    await message.channel.send('ここはTextChannelだよ')
```
